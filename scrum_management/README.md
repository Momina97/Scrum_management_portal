Scrum Sidebar Demo
==================

Installation:
1. Unzip into your Odoo addons directory or place the folder `scrum_management` inside addons.
2. Restart Odoo server.
3. Update Apps list and install "Scrum Sidebar Demo".
4. Open "Scrum Dashboard" from the Scrum menu.

Notes:
- Built for Odoo 18 web client (OWL-based). It's a minimal demo showing a collapsible left sidebar.
