Product / Inventory Requisition

A fully dynamic, configurable approval workflow for internal product and inventory requisitions. Supports Local & Foreign procurement modes, multi-level approvals, automated RFQ generation, and a clean portal interface for requesters.

Key Features

  • ✓ Configurable multi-step approval workflows per department or global fallback
  • ✓ Local vs Foreign requisition classification with currency auto-selection
  • ✓ Part numbers, estimated unit prices, and on-hand qty on requisition lines
  • ✓ One-click multi-vendor RFQ generation with Incoterm & payment term support
  • ✓ Full approval history log with timestamps and comments
  • ✓ Portal access for employees to submit and track their requisitions
  • ✓ Rejection flow with mandatory reason and reset-to-draft capability
  • ✓ Printable PDF report for every requisition

Workflow States

Draft → Submitted → In Approval → Approved → RFQ Created → Done  |  Rejected / Cancelled

Dependencies

base • crm • stock • purchase • mail • hr • portal • analytic
